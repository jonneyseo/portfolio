# John Seo — Portfolio

Personal portfolio website for John Seo, Digital Product Creator.

## Overview

A static HTML/CSS portfolio showcasing product design and UX work across enterprise and freelance projects.

## Structure

```
/               → Main portfolio (English)
/kr/            → Korean version
/resume.html    → Resume
/works.html     → Work index
/bcit/          → BCIT student project archive
```

## Stack

Plain HTML, CSS, no frameworks or build tools — just open `index.html` in a browser.
